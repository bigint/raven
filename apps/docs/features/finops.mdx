The FinOps module provides financial reporting and optimization insights for your AI spending.

## Cost Optimization

Raven analyzes your usage patterns and provides recommendations:

- **Model efficiency** — Identify opportunities to use cheaper models for simple tasks
- **Cache opportunities** — See how much you could save by enabling semantic caching
- **Routing optimization** — Suggestions for routing rules that could reduce costs
- **Unused providers** — Identify configured providers that aren't being used

## Reports

Generate financial reports for:

- **Cost breakdown** — Spending by provider, model, team, or key
- **Trend analysis** — Cost trends over time with projections
- **Budget utilization** — How close each budget is to its limit
- **Per-request economics** — Average cost per request by model

## Cost Allocation

Track costs by:

| Dimension | Description |
|-----------|-------------|
| **Organization** | Total organization spend |
| **Team** | Spend by team |
| **Virtual Key** | Spend by API key |
| **Model** | Spend by model |
| **Provider** | Spend by provider |

This enables chargeback and showback for internal cost allocation.
