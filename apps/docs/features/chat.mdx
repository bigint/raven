Raven includes a built-in chat interface for testing models, prompts, and configurations directly in the dashboard.

## Using the Chat Interface

Navigate to **Chat** in the dashboard to start a conversation.

### Features

- **Model selection** — Choose any model from your configured providers
- **Conversation history** — All conversations are saved with full message history
- **Token tracking** — See token counts and costs for each message
- **Streaming** — Real-time streamed responses

## Conversations

Each chat session is saved as a conversation with:

| Field | Description |
|-------|-------------|
| **Title** | Auto-generated or custom title |
| **Model** | The model used for the conversation |
| **Messages** | Full message history with roles |
| **Tokens** | Total input and output token counts |
| **Metadata** | Additional context and tags |

## Use Cases

- **Prompt testing** — Iterate on prompt templates before deploying
- **Model comparison** — Compare responses across different models
- **Debugging** — Reproduce and debug specific request patterns
- **Demo** — Showcase AI capabilities to stakeholders
