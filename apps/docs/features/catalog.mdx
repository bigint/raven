The catalog is a shared registry where team members can publish and discover reusable resources.

## Catalog Items

The catalog supports multiple resource types:

| Type | Description |
|------|-------------|
| **Models** | Curated model configurations with recommended settings |
| **Agents** | Pre-configured agent definitions |
| **Prompts** | Shared prompt templates |
| **MCP Servers** | Published MCP server connections |

## Publishing to Catalog

Any resource can be published to the organization catalog:

<Steps>
  <Step title="Navigate to the resource">
    Go to the resource you want to share (agent, prompt, etc.).
  </Step>
  <Step title="Click Publish to Catalog">
    Add a description and any usage notes.
  </Step>
  <Step title="Set visibility">
    Choose who can see and use the resource.
  </Step>
</Steps>

## Discovering Resources

Browse the catalog to find resources shared by your team:

- **Search** by name or description
- **Filter** by resource type
- **Sort** by popularity or recency

## Use Cases

- **Standardization** — Share approved model configurations across teams
- **Knowledge sharing** — Publish battle-tested prompt templates
- **Onboarding** — New team members can discover existing resources
- **Governance** — Approve and curate resources centrally
