## Update Provider

`PATCH /providers/:id`

Updates an existing provider configuration.

### Path Parameters

| Parameter | Description |
|-----------|-------------|
| `id` | Provider ID |

### Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `apiKey` | string | No | New provider API key |
| `name` | string | No | New display name |
| `isEnabled` | boolean | No | Enable or disable the provider |

### Example Request

```bash
curl -X PATCH http://localhost:3001/providers/prov_abc123 \
  -H "Authorization: Bearer rk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "isEnabled": false
  }'
```

### Example Response

```json
{
  "id": "prov_abc123",
  "provider": "openai",
  "name": "Production OpenAI",
  "isEnabled": false,
  "createdAt": "2025-01-15T10:00:00Z",
  "updatedAt": "2025-01-20T14:30:00Z"
}
```
