## Update Organization

`PATCH /settings`

Updates the organization's settings.

### Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | No | New organization name |
| `slug` | string | No | New URL slug |

### Example Request

```bash
curl -X PATCH https://api.raven.dev/settings \
  -H "Authorization: Bearer rk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme AI Corp"
  }'
```

### Example Response

```json
{
  "id": "org_abc123",
  "name": "Acme AI Corp",
  "slug": "acme-corp",
  "plan": "team",
  "updatedAt": "2025-01-20T14:30:00Z"
}
```
