## Create Prompt

`POST /prompts`

Creates a new prompt template with an initial version.

### Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | Unique prompt name |
| `content` | string | Yes | Initial version content |

### Example Request

```bash
curl -X POST https://api.raven.dev/prompts \
  -H "Authorization: Bearer rk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "name": "summarizer",
    "content": "You are a concise summarizer. Summarize the following text in 3 bullet points."
  }'
```

### Example Response

```json
{
  "id": "prompt_abc123",
  "name": "summarizer",
  "activeVersionId": "ver_xyz789",
  "versions": [
    {
      "id": "ver_xyz789",
      "content": "You are a concise summarizer. Summarize the following text in 3 bullet points.",
      "isActive": true,
      "createdAt": "2025-01-15T10:00:00Z"
    }
  ],
  "createdAt": "2025-01-15T10:00:00Z"
}
```
