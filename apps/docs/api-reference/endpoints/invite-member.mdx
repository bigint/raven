## Invite Member

`POST /teams/invite`

Sends an invitation email to join the organization.

### Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `email` | string | Yes | Email address to invite |
| `role` | string | Yes | Role: `admin`, `member`, or `viewer` |

### Example Request

```bash
curl -X POST http://localhost:3001/teams/invite \
  -H "Authorization: Bearer rk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "email": "carol@example.com",
    "role": "member"
  }'
```

### Example Response

```json
{
  "id": "inv_abc123",
  "email": "carol@example.com",
  "role": "member",
  "status": "pending",
  "createdAt": "2025-01-15T10:00:00Z",
  "expiresAt": "2025-01-22T10:00:00Z"
}
```

<Note>
  Invitations expire after 7 days. The invitee receives an email with a link to accept the invitation.
</Note>
