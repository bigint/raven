## List Budgets

`GET /budgets`

Returns all budgets for the organization.

### Example Request

```bash
curl -X GET http://localhost:3001/budgets \
  -H "Authorization: Bearer rk_live_..."
```

### Example Response

```json
[
  {
    "id": "bud_abc123",
    "name": "Monthly production limit",
    "amount": 500.00,
    "period": "monthly",
    "currentSpend": 142.50,
    "alertThreshold": 80,
    "createdAt": "2025-01-01T00:00:00Z"
  }
]
```
