## Delete Routing Rule

`DELETE /routing-rules/:id`

Permanently deletes a routing rule.

### Path Parameters

| Parameter | Description |
|-----------|-------------|
| `id` | Routing rule ID |

### Example Request

```bash
curl -X DELETE http://localhost:3001/routing-rules/rule_abc123 \
  -H "Authorization: Bearer rk_live_..."
```

### Example Response

```json
{
  "success": true
}
```
