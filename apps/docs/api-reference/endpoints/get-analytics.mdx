## Get Analytics

`GET /analytics`

Returns aggregated usage and cost analytics.

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `startDate` | string | Start date (ISO 8601) |
| `endDate` | string | End date (ISO 8601) |
| `groupBy` | string | Dimension: `model`, `provider`, `key`, `day` |

### Example Request

```bash
curl -X GET "http://localhost:4000/analytics?startDate=2025-01-01&endDate=2025-01-31&groupBy=model" \
  -H "Authorization: Bearer rk_live_..."
```
