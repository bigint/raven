## Delete Webhook

`DELETE /webhooks/:id`

Permanently deletes a webhook endpoint.

### Path Parameters

| Parameter | Description |
|-----------|-------------|
| `id` | Webhook ID |

### Example Request

```bash
curl -X DELETE http://localhost:4000/webhooks/wh_abc123 \
  -H "Authorization: Bearer rk_live_..."
```

### Example Response

```json
{
  "success": true
}
```
