## Create Budget

`POST /budgets`

Creates a new cost budget.

### Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | Budget name |
| `amount` | number | Yes | Budget limit in USD |
| `period` | string | Yes | `daily` or `monthly` |
| `alertThreshold` | number | No | Alert at this percentage (0-100) |

### Example Request

```bash
curl -X POST http://localhost:4000/budgets \
  -H "Authorization: Bearer rk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Monthly production limit",
    "amount": 500,
    "period": "monthly",
    "alertThreshold": 80
  }'
```

### Example Response

```json
{
  "id": "bud_abc123",
  "name": "Monthly production limit",
  "amount": 500.00,
  "period": "monthly",
  "currentSpend": 0,
  "alertThreshold": 80,
  "createdAt": "2025-01-15T10:00:00Z"
}
```
