## List Models

`GET /models`

Returns the model catalog with pricing and capability information.

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `provider` | string | Filter by provider |
| `capability` | string | Filter by capability |

### Example Request

```bash
curl -X GET http://localhost:3001/models \
  -H "Authorization: Bearer rk_live_..."
```
