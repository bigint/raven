## Delete Budget

`DELETE /budgets/:id`

Permanently deletes a budget.

### Path Parameters

| Parameter | Description |
|-----------|-------------|
| `id` | Budget ID |

### Example Request

```bash
curl -X DELETE http://localhost:4000/budgets/bud_abc123 \
  -H "Authorization: Bearer rk_live_..."
```

### Example Response

```json
{
  "success": true
}
```
