## Get Organization

`GET /settings`

Returns the current organization's details and settings.

### Example Request

```bash
curl -X GET http://localhost:3001/settings \
  -H "Authorization: Bearer rk_live_..."
```

### Example Response

```json
{
  "id": "org_abc123",
  "name": "Acme Corp",
  "slug": "acme-corp",
  "plan": "team",
  "createdAt": "2025-01-01T00:00:00Z"
}
```
