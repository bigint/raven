## Delete Guardrail

`DELETE /guardrails/:id`

Permanently deletes a guardrail rule.

### Path Parameters

| Parameter | Description |
|-----------|-------------|
| `id` | Guardrail ID |

### Example Request

```bash
curl -X DELETE http://localhost:3001/guardrails/guard_abc123 \
  -H "Authorization: Bearer rk_live_..."
```

### Example Response

```json
{
  "success": true
}
```
