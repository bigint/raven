## Get Request Logs

`GET /analytics/requests`

Returns detailed request logs with filtering and pagination.

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `limit` | integer | Number of results (default: 50, max: 100) |
| `offset` | integer | Pagination offset |
| `model` | string | Filter by model |
| `provider` | string | Filter by provider |
| `status` | integer | Filter by HTTP status code |
| `keyId` | string | Filter by virtual key ID |

### Example Request

```bash
curl -X GET "http://localhost:4000/analytics/requests?limit=10&model=gpt-4o" \
  -H "Authorization: Bearer rk_live_..."
```
