## List Virtual Keys

`GET /keys`

Returns all virtual keys for the authenticated organization.

### Example Request

```bash
curl -X GET http://localhost:4000/keys \
  -H "Authorization: Bearer rk_live_..."
```

### Example Response

```json
[
  {
    "id": "key_abc123",
    "name": "Production App",
    "prefix": "rk_live_abc1",
    "environment": "live",
    "rateLimitRpm": 100,
    "rateLimitRpd": 10000,
    "expiresAt": null,
    "createdAt": "2025-01-15T10:00:00Z"
  }
]
```

<Note>
  Only the key prefix is returned, never the full key. The full key is shown only once at creation time.
</Note>
