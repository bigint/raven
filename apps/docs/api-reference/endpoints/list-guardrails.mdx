## List Guardrails

`GET /guardrails`

Returns all guardrail rules for the organization.

### Example Request

```bash
curl -X GET http://localhost:4000/guardrails \
  -H "Authorization: Bearer rk_live_..."
```

### Example Response

```json
[
  {
    "id": "guard_abc123",
    "name": "Block PII",
    "type": "pii",
    "action": "block",
    "config": {
      "entities": ["email", "phone", "ssn"]
    },
    "isEnabled": true,
    "createdAt": "2025-01-15T10:00:00Z"
  }
]
```
