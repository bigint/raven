## Update Budget

`PATCH /budgets/:id`

Updates an existing budget.

### Path Parameters

| Parameter | Description |
|-----------|-------------|
| `id` | Budget ID |

### Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | No | New name |
| `amount` | number | No | New budget limit |
| `alertThreshold` | number | No | New alert threshold |

### Example Request

```bash
curl -X PATCH https://api.raven.dev/budgets/bud_abc123 \
  -H "Authorization: Bearer rk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 750
  }'
```

### Example Response

```json
{
  "id": "bud_abc123",
  "name": "Monthly production limit",
  "amount": 750.00,
  "period": "monthly",
  "currentSpend": 142.50,
  "alertThreshold": 80,
  "updatedAt": "2025-01-20T14:30:00Z"
}
```
