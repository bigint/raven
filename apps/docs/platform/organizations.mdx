Organizations are the top-level workspace in Raven. All resources -- providers, keys, budgets, guardrails, and team members -- belong to an organization. Every user has at least one organization, and can belong to multiple.

## Creating an Organization

An organization is created automatically when you sign up. You can create additional organizations from the dashboard.

<Steps>
  <Step title="Open Organization Selector">
    Click the organization name in the sidebar to open the selector.
  </Step>
  <Step title="Click Create Organization">
    Select **Create Organization** at the bottom of the dropdown.
  </Step>
  <Step title="Enter Details">
    Provide a name and URL slug for your new organization.
  </Step>
</Steps>

### Organization Properties

| Field | Description |
|-------|-------------|
| **Name** | Display name shown in the dashboard |
| **Slug** | URL-safe identifier (e.g., `acme-corp`) used in dashboard URLs |
| **Plan** | Subscription tier: Free, Pro, Team, or Enterprise |

## Organization Settings

Navigate to **Settings** to configure your organization:

### General

- **Name** -- Update your organization's display name
- **Slug** -- Change the URL-friendly identifier
- **Branding** -- Customize dashboard appearance

### Security

- **IP Allowlists** -- Restrict API access to trusted IPs (see [IP Allowlists](/security/ip-allowlists))

### Infrastructure

- **Custom Domains** -- Add custom domains for the API proxy (see [Custom Domains](/platform/custom-domains))
- **Cache** -- Enable or disable semantic response caching

### Danger Zone

- **Delete Organization** -- Permanently delete the organization and all its resources

## Member Management

### Inviting Members

<Steps>
  <Step title="Navigate to Team">
    Go to the **Team** section in the dashboard.
  </Step>
  <Step title="Click Invite">
    Enter the user's email address.
  </Step>
  <Step title="Select a Role">
    Choose from Owner, Admin, Manager, Developer, or Viewer (see [Roles & Permissions](/platform/roles)).
  </Step>
  <Step title="Send Invitation">
    The user receives an email invitation to join the organization.
  </Step>
</Steps>

### Managing Members

- **Change roles** -- Owners and admins can change any member's role
- **Remove members** -- Remove a member from the organization without affecting their account
- **Transfer ownership** -- Transfer the Owner role to another member

## Switching Between Organizations

If you belong to multiple organizations, switch between them using the organization selector in the sidebar. Your current organization determines which resources you see and manage.

<Note>
  Each organization has its own plan, billing, and resource limits. Switching organizations switches your entire context -- providers, keys, analytics, and settings.
</Note>

## Organization-Scoped Resources

All resources in Raven are scoped to an organization. There is no sharing of resources between organizations.

| Resource | Scoped To |
|----------|-----------|
| **Providers** | Organization |
| **Virtual Keys** | Organization (optionally team) |
| **Budgets** | Organization, team, or key |
| **Guardrails** | Organization |
| **Policies** | Organization |
| **Routing Rules** | Organization |
| **Analytics** | Organization |
| **Audit Logs** | Organization |
| **Webhooks** | Organization |
| **Custom Domains** | Organization |

## API Access

When making management API requests, specify your organization using the `X-Org-Id` header:

```bash
curl -X GET https://api.raven.dev/providers \
  -H "Authorization: Bearer rk_live_..." \
  -H "X-Org-Id: org_abc123"
```

For proxy requests (chat completions), the organization is determined automatically from the virtual key.

## Plan Limits by Organization

Each organization's plan determines its resource limits:

| Resource | Free | Pro | Team | Enterprise |
|----------|------|-----|------|------------|
| **Seats** | 1 | 1 | 30 | Unlimited |
| **Providers** | 2 | 10 | Unlimited | Unlimited |
| **Virtual Keys** | 3 | 20 | Unlimited | Unlimited |
| **Monthly Requests** | 10,000 | 500,000 | 2,000,000 | Unlimited |

See [Billing & Plans](/platform/billing) for the complete feature comparison.
